# Turf-Booking-System


A **web-based Turf Booking System** built using **PHP, MySQL, Bootstrap, HTML, and CSS**.
This application allows users to browse available turfs, book time slots, and manage their bookings, while turf owners can add and manage their turfs. The system also includes an admin dashboard for monitoring users, turfs, and bookings.

---

## 🚀 Features

### 👤 User

* Register and login securely
* Browse available turfs
* View turf details (location, price, image)
* Book available time slots
* View and cancel bookings

### 🏟 Turf Owner

* Add new turfs with images
* Manage existing turfs
* Edit or delete turf listings

### 🛠 Admin

* View all users
* View all turfs
* View booking records
* Dashboard overview of the system

---

## 🖥 Technologies Used

* **Frontend:** HTML, CSS, Bootstrap

---

## ⚙️ Installation Guide

1. Clone the repository
2. Place the project folder inside **htdocs** (XAMPP)
3. Start **Apache and MySQL** in XAMPP
4. Open **phpMyAdmin** and import `turf-booking.sql`
5. Run the project in your browser:

http://localhost/turf-booking
